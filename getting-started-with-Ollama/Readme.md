# Getting Started with Ollama

This repository contains example code for getting started with **Ollama**, a tool that lets you run open‑source large language models (like Llama 3, Mistral, etc.) locally on your machine. It demonstrates how to pull and run models, interact with them via the Ollama CLI, and connect them to Python applications.

## What this covers
- Installing and starting Ollama.
- Pulling and running open‑source LLMs (e.g., `ollama pull llama3`).
- Basic interaction with the model in the terminal.
- Simple Python integration (CLI + Python) to call models locally without relying on cloud APIs.

## How to use
1. Install Ollama from the official site and add it to your `PATH`.  
2. Run a model, e.g., `ollama run llama3`.  
3. Use the provided examples to call Ollama models from Python (can be extended with LangChain or direct REST calls).
