# Building a Basic LLM Application Using LCEL

This repository contains the implementation of a basic LLM application built using LangChain Expression Language (LCEL), as covered in the *Complete Generative AI Course with LangChain and Huggingface* by Krish Naik. It demonstrates how to compose prompts, models, and parsers into a clean, declarative pipeline using the `|` operator and core LangChain components.
