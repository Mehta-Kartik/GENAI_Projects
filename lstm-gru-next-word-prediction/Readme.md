# End‑to‑End Deep Learning Project with LSTM and GRU for Next‑Word Prediction

This repository contains an end‑to‑end deep learning project that uses LSTM and GRU networks to predict the next word in a sequence. It includes data preprocessing, model building, training, and evaluation, demonstrating how RNN variants can be applied to natural language generation and sequence‑prediction tasks from the "Complete Generative AI Course with Langchain and Huggingface" by Krish Naik.
