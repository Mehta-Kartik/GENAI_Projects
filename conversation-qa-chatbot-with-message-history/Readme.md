# Conversation QA Chatbot with Message History

This repository contains an implementation of a Conversational Q&A chatbot that maintains message‑based conversation history, using LangChain components as covered in the *Complete Generative AI Course with LangChain and Huggingface* by Krish Naik. It demonstrates how to structure prompts, store human/AI messages, and inject chat history into the LLM so the bot can answer follow‑up questions contextually.
